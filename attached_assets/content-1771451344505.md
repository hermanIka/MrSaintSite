# Guide de référence des API

La plateforme d'intégration de MaishaPay permet aux développeurs d'intégrer MaishaPay à leur plateforme afin de recevoir des paiements via leurs applications mobiles ou leurs sites web.

##### Mpay REST API DOC

MaishaPay fournit une API REST vous permettant d'automatiser le traitement des paiements avec n'importe lesquels de vos applications métier.


##### Checkout

Checkout crée une page de paiement sécurisée hébergée par MaishaPay qui vous permet de collecter rapidement les paiements. Il fonctionne sur tous les appareils et est conçu pour augmenter votre vente


##### Mobile Money

Grâce à l'API Mobile Money, il est possible de collecter des paiements dans plusieurs pays africains, tels que la République Démocratique du Congo (Kinshasa), la République du Congo (Brazzaville), le Cameroun, la Côte d'Ivoire, l'Égypte et l'Éthiopie.


##### Woocommerce

Recevez facilement des paiements dans le monde entier avec le plugin de paiement Maishapay WooCommerce.


##### Transfert B2C

Un API de transfert B2C (Business to Consumer) est une interface de programmation d'applications qui permet aux entreprises (Business) d'envoyer des paiements ou des transferts d'argent directement aux consommateurs (Consumer).


##### Collecte via Carte

Grâce à l'API de collecte de paiements MaishaPay, vous pouvez collecter des paiements du monde entier via des canaux tels que Visa, MasterCard, American Express, UnionPay, le tout avec une sécurité 3D Secure.


##### Crypto Money

API de collecte de crypto-monnaies permet aux entreprises ou développeurs d'intégrer des fonctionnalités pour recevoir des paiements ou des dons en crypto-monnaies (comme le Bitcoin, l'Ethereum, etc.) directement dans leurs applications ou sites web.


##### Airtimes

L'API Airtimes de Maishapay permet aux développeurs d'intégrer des fonctionnalités de recharge de crédit téléphonique (recharge de cartes prépayées) directement dans des applications ou sites web, et connecte plus de 50 pays à travers le monde.


##### Paytv

L'API PayTV permet aux développeurs d'intégrer des fonctionnalités de paiement pour les services de télévision payante directement dans des applications ou sites web. Elle permet aux utilisateurs de payer leurs abonnements ou d'acheter des contenus à la demande (comme Canal+, Dvia des paiements en ligne).
